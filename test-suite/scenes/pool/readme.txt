We provide the Swimming pool scene as a part of a demo package that implements the importance sampling technique presented 
in our paper "On-line Learning of Parametric Mixture Models for Light Transport Simulation".

--------------------------------------------------------------------------------

Permission is granted to use the scene for light transport research/publication, given that:
- The scene source files are NOT made available to download at any public online source
- You can use the scene for any purpose provided that we are given credit.

--------------------------------------------------------------------------------

Provided formats: 
- Mitsuba 0.5.0 .xml file + serialized geometry (contains a special material named "coronamtl" implemented as a Mitsuba plug-in in the mitsuba_050/coronamtl.cpp file)
